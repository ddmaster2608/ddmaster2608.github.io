什么是sql注入?

原理如下：

假设后端查询语句为：

SELECT * FROM users WHERE id = '$id';

其中 $id 是用户输入的参数。

当用户输入 ' or '1'='1 时，参数替换后查询变为：

SELECT * FROM users WHERE id = '' or '1'='1';

注意到' or '1'='1中第一个’和最后一个’是为了闭合后端原本存在的’号

所以这题payload就有了：/?id=' or '1'='1

也可以/?id=' or 1=1 -- +其中–+起到注释后文的作用
